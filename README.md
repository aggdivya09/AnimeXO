# AnimeXO
An anime-themed Tic-Tac-Toe game developed using Python, Tkinter, and Pillow.
It combines the classic Tic-Tac-Toe experience with anime-inspired visuals, unlockable character themes, intelligent AI gameplay, and a modern responsive interface.

---

## 📌 Overview

AnimeXO is a desktop game designed for anime fans who want a unique twist on the traditional Tic-Tac-Toe game.

Players can choose from multiple anime-inspired themes, challenge a smart AI opponent, unlock new themes using earned coins.
This project demonstrates practical implementation of Python GUI development, image processing, game logic, responsive design, and AI algorithms.

---

## 🎮 Features

- Single Player Mode against AI
- Two Player Local Multiplayer
- Smart AI using Minimax Algorithm
- Anime-inspired themes and visuals
- Unlockable theme system
- Coin reward system
- Responsive and resizable interface
- Dynamic background and button scaling
- Persistent game progress using JSON
- Theme-specific game assets
- Interactive menu navigation
- Modern anime-style user experience

## 🤖 AI System

AnimeXO features an intelligent AI opponent powered by the **Minimax Algorithm**.

### AI Features

- Strategic decision making
- Near-optimal gameplay
- Win prevention logic
- Dynamic move evaluation
- Balanced difficulty through randomized moves

The AI provides a challenging experience while remaining enjoyable for casual players.

---

## 🪙 Coin & Unlock System

The game rewards players for winning matches.

### Rewards

- Win a game → +20 Coins
- Unlock a new theme → -100 Coins

All progress is automatically saved and restored when the game is reopened.
